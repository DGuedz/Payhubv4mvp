This is a code bundle for Design DApp Mobile PAYHUB. The original project is available at https://www.figma.com/design/x2bTedifXXqT4ysP4INEHU/Design-DApp-Mobile-PAYHUB.

## Production

- URL: https://payhubv4mvp.vercel.app
- Edge cache enabled and HTTPS by default

## Monitoring

- Web Analytics and Speed Insights can be enabled in the Vercel dashboard
- Packages integrated: `@vercel/analytics`, `@vercel/speed-insights`

## Running the code

- Install dependencies: `npm i`
- Development: `npm run dev`
- Production build: `npm run build` (outputs to `build/`)
