### Features do Dashboard
- ✨ Pagamento PIX com QR dinâmico e callback automático
- ✨ Escrow RLUSD com wizard de 4 passos
- ✨ Yield automático (5-8% APY)
- ✨ Roteamento AMM com pathfind
- ✨ Auditoria com exportação CSV (CARF/OCDE)
- ✨ SDK modular com retry exponencial
- ✨ Toast notifications globais
- ✨ Navegação omnicanal (mobile/desktop)

### Componentes Principais
- PaymentPix - Modal de pagamento com 4 steps
- DashboardHome - Widget principal
- DashboardNav - Navegação mobile tab bar + desktop sidebar
- EscrowWizard - Wizard completo de Escrow
- YieldCard - Ativação de yield
- AMMCard - Consulta AMM pathfind
- AuditTable - Tabela + exportação CSV
- Toast - Sistema de notificações

### SDK Features
- Base URL inteligente (auto-detecta origin)
- Retry com backoff exponencial
- currencyHex() browser-compatible
- Módulos: trustline, escrow, amm, yield, compliance, security

### Segurança & Compliance
- XRPL_SEED isolada no backend (KMS/ENV)
- JWT curto obrigatório
- Auditoria sem PII (apenas txHash/sequence)
- Banner CARF/OCDE (IN RFB nº 2.291/2025)
- Cookie consent GOV.BR
